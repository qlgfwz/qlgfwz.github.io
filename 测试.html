API接口说明

接口1：https://pan.szfx.top/api.php

请求方式：POST multipart/form-data

请求参数： 

字段名       是否必填   示例值       描述
file         是        multipart格式 文件
show         否        1            是否首页显示（默认是）
ispwd        否        0            是否设置密码（默认否）
pwd          否        123456       下载密码（默认留空）
format       否        json         返回格式（json/jsonp/form，默认json）
urlbackurl   否        http://...   上传成功跳转地址（仅form格式有效）
callback     否        callback     回调函数（仅jsonp格式有效）

返回参数（JSON格式）： 

{
    "code": 0,
    "msg": "上传成功！",
    "hash": "f1e807cb0d6ba52d71bdb02864e6bda8",
    "name": "exapmle1.jpg",
    "size": 58937,
    "type": "jpg",
    "downurl": "http://.....",
    "viewurl": "http://....."
}
